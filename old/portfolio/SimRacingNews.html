Sim Racing News (Android app)

This was my second Android Mobile application(App). Developed in the official Android SDK which is based on the Java programming language. So therefore a bit more complicated. The main technologies used in this app are DOM Document Object Model and the inbuilt SQL database SQLite3. Other technologies used are InputStream and WebView.

I'm a big fan of PC racing games. It has a big online community with some useful news feeds. I decided to make an app that takes these feeds and puts them into one app. It also allows you to send an email reminder of the mods you are interested in for later downloading. I am proud to say it won App of the week at www.racesimcentral.com
