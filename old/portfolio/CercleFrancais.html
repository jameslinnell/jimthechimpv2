Cercle Francais de Winchester

My first drupal website project was to help modernise a local registered charity's website free of charge. The Cercle Francais de Winchester is an association for french speakers in Winchester. It holds lectures and events throughout the year and so wanted a content management system to be able to update the calendar events. The website also needed a more moden design to help attract younger members. This is currently an ongoing project.
